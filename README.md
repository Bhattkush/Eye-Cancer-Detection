# Eye-Cancer-Detection

An AI-powered web application that detects possible eye cancer from uploaded eye images using a deep learning model built with TensorFlow and deployed using Streamlit.

## 🚀 Features

- Upload eye images (JPG, PNG, JPEG)
- AI model predicts cancer presence
- Instant result display:
  - ⚠️ Eye Cancer Detected
  - ✅ Normal Eye
- Simple and user-friendly UI

---

## 🛠️ Tech Stack

- **Python**
- **TensorFlow / Keras**
- **OpenCV**
- **NumPy**
- **Streamlit**
